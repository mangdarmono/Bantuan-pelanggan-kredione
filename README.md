# KrediOne - Layanan Bantuan Finansial

Landing page modern untuk layanan bantuan finansial KrediOne. Dibangun dengan Vite + React + Tailwind CSS.

## Tech Stack

- Vite - Build tool
- React 18 - UI library
- Tailwind CSS - Styling
- Lucide React - Icons

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Features

- Responsive design (mobile, tablet, desktop)
- SEO optimized with meta tags and Open Graph
- Schema.org structured data
- Smooth scroll navigation
- WhatsApp integration
- Modern premium UI
